## Requirements
Node.js - 12.19.0(https://nodejs.org/en/)  
MongoDB Atlas(https://www.mongodb.com/cloud/atlas)


## Development environment  
```bash
git clone git@github.com:ProfesorSi/halal-restorani-backend.git  
cd halal-restorani-backend  
npm install  
npm install nodemon  
nodemon server.js
```
